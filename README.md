# CleanCloud

## Node Setup
* Run `npm install` to install the project dependencies

## Development Servers
* Run `npm run dev` to run the Angular server and Node.js server
* Run `npm run mock` to run the Angular server and mock Node.js server
* Port 4200

## Production Server
* Run `npm start`
* Port 3000
